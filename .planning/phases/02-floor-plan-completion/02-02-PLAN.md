Add a `useEffect` that detects users transitioning to `offline` status:
```typescript
useEffect(() => {
  const prevUsers = prevUsersRef.current;
  const currentIds = new Set(users.map(u => u.id));
  const prevIds = new Set(prevUsers.map(u => u.id));

  // Detect users who are still in the list but changed to offline status
  users.forEach(user => {
    if (user.status === 'offline' && !exitingUsersRef.current.has(user.id)) {
      const prevUser = prevUsers.find(p => p.id === user.id);
      if (prevUser && prevUser.status !== 'offline') {
        // User just went offline -- start fade-out
        exitingUsersRef.current.set(user.id, Date.now());
        setExitingUserIds(prev => new Set([...prev, user.id]));
        // Remove from DOM after fade completes (3 seconds)
        setTimeout(() => {
          exitingUsersRef.current.delete(user.id);
          setExitingUserIds(prev => {
            const next = new Set(prev);
            next.delete(user.id);
            return next;
          });
        }, 3000);
      }
    }
  });

// Also detect users who disappeared from the list entirely after an offline transition
prevIds.forEach(id => {
    if (!currentIds.has(id) && !exitingUsersRef.current.has(id)) {
      // User removed from list -- they're already gone, no fade needed
    }
  });

  prevUsersRef.current = [...users];
}, [users]);
```

**1. Extend `src/hooks/useUserPresence.ts` with beforeunload handler and enhanced presence detection:**

Add a `beforeunload` event listener that records disconnect time immediately when the tab is intentionally closed, then sends best-effort offline cleanup to the location API:
```typescript
// Inside the hook, after the existing useEffect blocks:
useEffect(() => {
  if (!currentUserId) return;

  const handleBeforeUnload = () => {
    // Save disconnect timestamp for grace period (FLOR-04 will use this)
    localStorage.setItem('vo-disconnect-timestamp', Date.now().toString());
    // Best-effort: notify server of departure using sendBeacon for reliability
    const payload = JSON.stringify({ userId: currentUserId, spaceId: null, offline: true });
    if (navigator.sendBeacon) {
      navigator.sendBeacon('/api/users/location', payload);
      return;
    }
    void fetch('/api/users/location', {
      method: 'POST',
      body: payload,
      headers: { 'Content-Type': 'text/plain;charset=UTF-8' },
      keepalive: true,
    });
  };

  const handleVisibilityChange = () => {
    if (document.visibilityState === 'hidden') {
      localStorage.setItem('vo-disconnect-timestamp', Date.now().toString());
    }
  };

  window.addEventListener('beforeunload', handleBeforeUnload);
  document.addEventListener('visibilitychange', handleVisibilityChange);

  return () => {
    window.removeEventListener('beforeunload', handleBeforeUnload);
    document.removeEventListener('visibilitychange', handleVisibilityChange);
  };
}, [currentUserId]);
```
