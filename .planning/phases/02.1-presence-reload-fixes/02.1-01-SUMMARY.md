## Deviations from Plan

### Auto-fixed Issues

**1. [Rule 1 - Bug] Avoided treating automatic placement as a manual click**
- **Found during:** Task 1 (Add saveLastSpace to automatic placement success paths)
- **Issue:** Calling the existing `saveLastSpace(spaceId)` directly from automatic placement would set `manualChangeRef`, which is reserved for user clicks and could incorrectly suppress the next auto-placement effect.
- **Fix:** Added `SaveLastSpaceOptions.markManualChange` with a default of `true`; automatic paths call `saveLastSpace(..., { markManualChange: false })`.
- **Files modified:** `src/hooks/useLastSpace.ts`, `src/components/floor-plan/floor-plan.tsx`
- **Verification:** Type-check, reconnection grace tests, default-space assignment test command, and users-location route regression tests.
- **Committed in:** `d013727` and `8e4690a`
